# TTrYs

A terminal-based Tetris game

![](assets/ttrys.png)

## Running the game

```shell
cargo run
```

## Playing the game

| Key    | Map
|--------|-------
| a      | move piece left
| s      | "soft drop" down
| d      | move piece right
| Left   | rotate piece counterclockwise
| Right  | rotate piece clockwise
| Space  | pause
